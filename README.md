# Observatorio San Marcos

Portal ciudadano de transparencia de la **Universidad Nacional Mayor de San Marcos** (UNMSM, pliego 510). Solo datos públicos.

**Live:** https://unimauro.github.io/observatorio-sanmarcos/

Módulos: presupuesto y ejecución (MEF/SIAF), en qué se gasta cada sol, proveedores y dueños (OECE/CONOSCE), investigación (OpenAlex — 44k publicaciones, la más productiva del país), gasto↔valor, personal (Portal de Transparencia). Nunca nombres de estudiantes.

Hermano de [Observatorio UNI](https://unimauro.github.io/observatorio-uni/) y [ENSAD](https://unimauro.github.io/observatorio-ensad/). No es un portal oficial de la UNMSM. Correcciones: carlos@cardenas.pe
